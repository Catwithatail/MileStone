Job Description

We are looking for an experienced Frontend Developer to join our growing team. The ideal candidate will have strong experience with React, TypeScript, and NextJS, and be passionate about creating clean, efficient, and maintainable code.


As a Senior Frontend Developer, you will be responsible for designing and implementing user interface components using React.js practices and workflows. You will work closely with the design team to ensure the technical feasibility of UI/UX designs, and collaborate with back-end developers to build the application.


Responsibilities:

  • Develop new user-facing features using React.js
  • Build reusable components and front-end libraries for future use
  • Translate designs and wireframes into high-quality code
  • Optimize components for maximum performance across devices and browsers
  • Coordinate with various teams working on different layers of the infrastructure

Requirements:

  • 5+ years of experience in front-end development
  • 3+ years of experience with React.js
  • Strong proficiency in JavaScript, TypeScript, and HTML/CSS
  • Experience with popular React workflows (Redux, Next.js)
  • Experience with responsive and adaptive design
  • Good understanding of asynchronous programming
  • Familiarity with Git and version control
Technical Skills:
React TypeScript NextJS Redux HTML5 CSS3 REST APIs Git Jest
